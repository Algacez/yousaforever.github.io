注意事项

Github图床 存储的图片总大小不能超过 1G，否则会有人工审核仓库内容，如果发现用来做图床，轻则删库，重则封号。因此为了安全起见，建议在 1G 之前就换个仓库，反正可以创建无数个仓库嘛。

jsDelivr + Npm 依然有 100MB 的包大小限制，但是 NPM 有丰富的国内节点。
